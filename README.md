# **FlutterWanAndroid**

Flutter练手项目，基于WanAndroid的Api,界面仿照WanAndroid官网客户端


### 模块

项目整体按照模块化划分，基于Get状态管理路由框架进行开发。

目前广播模块只做了列表页面，收藏模块也都没有做，后续有时间慢慢迭代
